### 🧑‍💻 About Me

- 🔭 I build **scalable backends** and **interactive frontends** — full-stack, end to end.
- 🏡 I run a **self-hosted homelab** (Docker, reverse proxies, automation) for fun and learning.
- 🌱 Always sharpening my skills across the JVM, Go, and modern web frameworks.
- 🧪 I work **test-first** — TDD & DDD, Agile/Scrum, and CI/CD pipelines.
- 🤖 I make **advanced, professional use of Claude** — building agents, subagents, custom skills, and hooks to automate real engineering workflows.
- 💬 Ask me about **Java / Spring**, **Go**, **Docker**, **MQTT**, or **self-hosting**.
- ⚡ Fun fact: I'm an **Arctic Code Vault Contributor** ❄️.
